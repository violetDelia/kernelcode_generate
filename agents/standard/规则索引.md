# 规则索引

## 文件说明
- 功能说明：提供 `agents/standard` 的入口索引，减少重复阅读。
- 使用示例：不知道先看哪份规则时，从本文件开始。

## 使用顺序
- 新会话先读本索引，再按当前角色读取个人提示词和对应标准文档。
- 角色名单、角色名和 session 真源以 `agents/codex-multi-agents/agents-lists.md` 为准；旧同名 session 或备份目录只作历史资料。
- 追溯上下文优先读取 `agents/codex-multi-agents/log/task_records/` 与当前仓库仍保留的任务记录。
